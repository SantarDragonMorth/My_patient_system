# My_patient_system

Данный репозиторий содержит программу для учёта и анализа данных о пациентах, которая позволяет добавлять, редактировать, удалять пациентов и их данные (ФИО, возраст, пол, рост, вес, ИМТ), а также получать о них сводную статистику.

## Установка и запуск

Проект устанавливается путём *клонирования* данного репозитория на свой компьютер:

    `git clone https://github.com/SantarDragonMorth/My_patient_system.git /home/username`
    
Для **запуска программы** выполните в терминале в папке */src* склонированного репозитория следующую команду:

    `python main.py`

Программа запустится и откроется окно с таблицей по пациентам. Кнопки взаимодествия с таблицей расположены внизу.

## Примеры использования

### Добавление пациента

1. Нажмите кнопку "Добавить"
2. Заполните форму о пациенте: ФИО, возраст, пол, рост, вес
3. Нажмите "Сохранить"

### Редактирование пациента

1. Выберите нужного пациента в таблице
2. Нажмите кнопку "Редактировать"
3. Внесите изменения в форму о пациенте
4. Нажмите "Сохранить"

### Удаление пациента

1. Выберите нужного пациента в таблице
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление пациента

### Получение сводной статистики

1. Нажмите кнопку "Статистика" для отображения аналитических графиков
2. Откроется окно со следующими графиками:
    - Распределение пациентов по полу

    - Распределение пациентов по возрасту

    - Распределение ИМТ с учётом пола

    - Зависимость ИМТ от возраста
    
### Генерация тестовых данных

1. Нажмите кнопку "Созд. данные"
2. В таблицу добавится 10 пациентов со случайными данными

## Внесение изменений в проект

Для **внесения изменений** в проект:
1. Форкните репозиторий.
2. Создайте ветку для изменений:

    `git checkout -b "my_features"`
    
3. Внесите изменения в файлы и закоммитьте их:

    `git commit -am "Added new feature"`
    
4. Запушьте изменения в Вашу ветку:

    `git push origin my_features`
    
5. Создайте пулл-реквест в основной репозиторий.
